# Continuous AI Chat Bot

A Django-based AI chat application that provides one continuous, never-ending conversation with DeepSeek AI, synchronized across all your devices.

# Features

- **One Continuous Conversation**: No more starting new chats - pick up exactly where you left off
- **Cross-Device Sync**: Your conversation follows you across all devices
- **DeepSeek AI Integration**: Powered by advanced AI for intelligent responses
- **Real-time Updates**: Messages sync automatically between devices
- **Clean, Responsive UI**: Built with Bootstrap for mobile and desktop

# Tech Stack

- **Backend**: Django 5.2
- **Frontend**: Bootstrap 5, Vanilla JavaScript
- **Database**: SQLite (development), PostgreSQL (production ready)
- **AI**: DeepSeek API
- **Deployment**: Ready for Railway/Render/Heroku
